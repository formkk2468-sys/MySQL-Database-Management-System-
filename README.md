# MySQL Database Management System 

## 📝 รายละเอียดโปรเจกต์
รายงานฉบับนี้เป็นส่วนหนึ่งของการศึกษาเรื่องการเขียนโปรแกรมและการประยุกต์ใช้โปรแกรม **MySQL** เพื่อจัดการระบบฐานข้อมูลการสั่งซื้อและการเคลมสินค้า โดยใช้ข้อมูลจำลองจากการสำรวจของบริษัท SAMSUNG เพื่อศึกษาความสัมพันธ์ของข้อมูลและฝึกการใช้คำสั่ง SQL ในการเรียกดูข้อมูลที่เป็นประโยชน์ต่อการวางแผนทางวิศวกรรม

## 📊 ER-Model
โครงสร้างความสัมพันธ์ของข้อมูลประกอบด้วย 5 ตารางหลัก:
1. **Product**: ข้อมูลสินค้า (รหัส, ชื่อ, ราคา, รุ่น)
2. **Customer**: ข้อมูลลูกค้า (รหัส, ชื่อ, ที่อยู่, เบอร์โทร)
3. **Employee**: ข้อมูลพนักงาน (รหัส, ชื่อ, ตำแหน่ง, วันเริ่มงาน/หมดสัญญา)
4. **Purchase**: ข้อมูลการสั่งซื้อ (เชื่อมโยงสินค้า ลูกค้า และพนักงาน)
5. **Claim**: ข้อมูลการเคลมสินค้า (สาเหตุการเคลม, ประเภทการเปลี่ยนสินค้า)
